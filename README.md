# SiteProfile - Perfil Discord

## Funcionalidades

- Página inicial com "click para entrar..."
- Página de perfil com:
  - Informações do perfil (nome, handle, badges)
  - Redes sociais (Instagram, TikTok, Spotify, Discord)
  - Status do Spotify (música atual tocando)
  - Seção Court Buxx com estatísticas

## Como executar

1. Instale as dependências:
```bash
npm install
```

2. Execute o servidor de desenvolvimento:
```bash
npm run dev
```

3. Abra [http://localhost:3000](http://localhost:3000) no seu navegador.

## Estrutura do Projeto

- `app/page.tsx` - Página inicial
- `app/profile/page.tsx` - Página de perfil
- `app/globals.css` - Estilos globais
- `app/*/page.module.css` - Estilos modulares por página

## Tecnologias

- Next.js 14
- React 18
- TypeScript
- CSS Modules
